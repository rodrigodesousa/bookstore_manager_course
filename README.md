# book-store-manager-api
## Demo para testes
```
https://book-store-manager-curso.herokuapp.com/
```
## Requisitos:
```
Java 11 ou superior
Spring Boot 2.4
Maven 3
```
## Utilizar IDE Eclipse ou Intellij para executar projeto ou as linhas de comando abaixo

## Instalação de dependencias
```
mvn install
```
## Executar Projeto
```
mvn spring-boot:run
```
